# Memory Game

A two-player memory card game built with Vite, TypeScript and SCSS.

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- **Vite** — Development server and build tool
- **TypeScript** — Type-safe JavaScript
- **SCSS** — 7-1 architecture pattern

## Features

- Two-player mode (Blue vs Orange)
- Multiple board sizes (4x4, 6x4, 6x6)
- Theme selection with different card motifs
- Card flip animations
- Score tracking and winner announcement
